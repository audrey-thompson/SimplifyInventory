The Simplify Inventory App is an app which I started creating in my Android Development course earlier this year. The complete idea is that it was a part of a system which would help families or individuals keep track of their home inventory. Users would receive two RFID receivers (one for the door where items enter the house and one for the door where they are brought out to the garbage) which would detect items as they entered and exited the home to keep track of the household’s inventory. Users would have the option to order Simplify branded products which came ready to work with the app or Simplify RFID sticker tags to add to their own personal items. There are also ways to change the inventory counts manually from within the app.  

I selected this artifact for inclusion in my ePortfolio because the idea was fully my own, and I find the idea very interesting and exciting. I think that there is a lot of room within the system for adding features, so it’s a canvas for the many skills I want to practice. As of now, I don’t think too many skills and abilities are apparent. I spent more time in the class planning and designing than actually developing the app. I’m excited to add accounts and complete the development according to the design I already have. I would also love to throw in some kind of ML functionality if I have time (I’ve taken a couple of classes that touched on these ideas which were very cool). Maybe after some time, I’ll even invest in some RFID receivers (though that would be outside the scope of this course).
